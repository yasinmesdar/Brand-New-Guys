Welkom

Welkom op mijn site. Ik heb deze website gemaakt naar aanleinding van het gesprek over een eventuele werkplek bij Brand New Guys. Op deze website zijn de opdrachten te vinden die ik heb meegekregen naar aanleiding van het gesprek.

Door de opdrachten te maken kan ik laten zien over welke skills ik beschik en bewijzen of ik een geschikte kandidaat ben als webdevelopper bij Brand New Guys. De eerste opdracht is het maken van een "Hello World" en de tweede is het maken van een post request naar brandnewguys.co/.netlify/functions/heimdall.

Voor elke opdracht heb ik een eigen pagina op de website gemaakt. Zoals gezegd is de eerste opdracht het maken van een "Hello world" in een taal naar keuze. Ik heb gekozen voor de talen PHP, Javascript en HTML. Op de pagina "opdracht 1" heb ik een textvak gemaakt met daarin de code die ik heb gebruikt voor het schrijven van "Hello world" in de verschillende talen. Verder is onder het textvak de resultaten te zien van de codes.

Voor de tweede opdracht heb ik een formulier gemaakt. In het formulier kun je de gevraagde gegevens invullen. Vervolgens kun je door op de knop "Submit" te drukken de gegevens verzenden. Voor het verzenden van de gegevens wodt een post method gebruikt. De post method zorgt ervoor dat de ingevulde data naar brandnewguys.co/.netlify/functions/heimdall wordt verstuurd.
